# TN-SCXD 攀岩墙定制系统 - 部署说明

## 📋 项目概述

这是一个基于 Next.js 15.3.3 的攀岩墙定制系统，包含以下主要功能：

### ✨ 核心功能
1. **产品管理系统**
   - 产品分类管理（攀岩板材、岩点、五金配件、复合板）
   - 产品启用/禁用功能
   - 基于客户类型的产品筛选

2. **客户管理系统**
   - 客户分类（未分类、OEM客户、品牌客户、工程客户）
   - 客户信息管理
   - 权限控制

3. **订单管理系统**
   - 订单创建和状态管理
   - 价格折扣功能
   - 订单打印（无价格，含产品图片）
   - 自动合同生成

4. **管理后台**
   - 全面的管理界面
   - 数据导出功能
   - 安全管理

## 🚀 部署步骤

### 1. 环境要求
- Node.js 18+ 
- npm 或 yarn
- 现代浏览器

### 2. 项目安装
```bash
# 1. 解压项目文件
unzip TN-SCXD-项目包_*.zip

# 2. 进入项目目录
cd TN-SCXD-1.0

# 3. 安装依赖
npm install

# 4. 构建项目
npm run build

# 5. 启动生产服务器
npm start
```

### 3. 开发环境启动
```bash
# 开发环境启动（热重载）
npm run dev
```

## 📁 项目结构

```
src/
├── app/                    # Next.js App Router 页面
│   ├── admin/             # 管理后台
│   ├── products/          # 产品中心
│   ├── orders/            # 订单管理
│   ├── cart/              # 购物车
│   └── checkout/          # 结账页面
```

## 🔧 技术栈

- **前端框架**: Next.js 15.3.3 (App Router)
- **UI 组件**: shadcn/ui + Tailwind CSS
- **状态管理**: Zustand
- **类型检查**: TypeScript
- **图标**: Lucide React
- **通知**: Sonner Toast
- **打印**: react-to-print
- **PDF生成**: jsPDF + html2canvas
- **Excel处理**: xlsx

## 🛠 功能详细说明

### 订单打印功能
- **特点**: 不显示价格信息，只显示客户编号
- **包含**: 产品图片、产品详情、客户编号
- **格式**: 支持直接浏览器打印

### 合同生成功能
- **自动化**: 根据订单内容自动生成合同条款
- **内容包含**: 
  - 产品详情清单
  - 交货信息
  - 客户联系方式
  - 标准合同条款
  - 生产备注
- **输出**: PDF格式下载

### 客户分类系统
- **未分类**: 新注册客户默认分类
- **OEM客户**: 代工客户，特定产品权限
- **品牌客户**: 品牌方客户
- **工程客户**: 工程项目客户

### 产品筛选机制
- 根据客户类型自动筛选可见产品
- 管理员可配置产品的目标客户群体
- 支持产品启用/禁用控制

## 🔍 故障排除

### 常见问题

1. **端口占用**
   ```bash
   # 如果 3000 端口被占用，Next.js 会自动选择其他端口
   # 或手动指定端口
   npm run dev -- -p 3001
   ```

2. **构建失败**
   ```bash
   # 清除缓存并重新安装
   rm -rf node_modules .next
   npm install
   npm run build
   ```

3. **样式问题**
   - 确保 Tailwind CSS 正常加载
   - 检查浏览器缓存
